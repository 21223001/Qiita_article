### **補数とは？**

補数（ほすう）は、ある数を基準の値に達するようにするための数のことです。
主に **減算を加算に変換** するために使われます。



### **何のためにあるのか？**
コンピュータでは **減算（引き算）を加算（足し算）だけで計算する** ために補数が使われます。
加算だけにすることで **回路を簡単にする** ことができ、処理が高速化されます。



### **実際に使われている例**
- **コンピュータの演算**  
  - **負の数の表現** に **2の補数** を使う
  - CPU での **減算の処理**
- **デジタル回路**  
  - ALU（算術論理演算装置）で加算・減算を統一する
- **計算機数学**  
  - 十進補数（9の補数や10の補数）を使って筆算の引き算を簡単にする



### **補数の種類と計算方法**

#### **1. n進数の (n-1) の補数**
- 基数 \( n \) の **(n-1)の補数** とは、各桁を **n-1 から引いた数** です。
- 例えば **10進数なら「9の補数」**、**2進数なら「1の補数」** になります。

**例1: 10進数 (9の補数)**
```
999 - 456 = 543
```
**9の補数: 543**

**例2: 2進数 (1の補数)**
```
1010_2 → 0101_2
```
**1の補数: 0101**



#### **2. n進数の nの補数**
- **(n-1) の補数に 1 を足す** と **nの補数** になる。
- 例えば **10進数なら「10の補数」**、**2進数なら「2の補数」** になります。
- これを使うと、減算を加算に変えられる。

**例1: 10進数 (10の補数)**
```
543 + 1 = 544
```
**10の補数: 544**

**例2: 2進数 (2の補数)**
```
0101_2 + 1 = 0110_2
```
**2の補数: 0110**



#### **補数を使った減算の例**
コンピュータでは、**減算を加算に変える** ために **2の補数** を使う。

例えば、
```
5 - 3
```
を **2進数で計算する**。

1. **5 を 2進数で表す:**
```
5_{10} = 0101_2
```

2. **3 を 2進数で表す:**
```
3_{10} = 0011_2
```

3. **3 の 2の補数を求める:**
```
1の補数:  1100_2
2の補数:  1101_2
```

4. **加算を行う:**
```
  0101_2
+ 1101_2
---------
 10010_2  (桁あふれは無視)
```
→ `0010_2 = 2_{10}` （答え）

### **ポイント**
- 減算を **「2の補数を足す」だけ** で処理できる
- **桁あふれは無視** することで、正しい答えが得られる



#### **覚え方**
- **1の補数** → **ビット反転（0⇔1）**
- **2の補数** → **1の補数 + 1**
- **減算は「2の補数を足す」**

これを押さえておけば、補数の考え方がしっかり理解できる
